JavsScript Password Strength

Check Password Strength

  • The user is presented a form asking for the user to create a password and then re-enter the same password.
  • As the user enters in the first password, the strength (length only) is checked with each downward key press. Based on the length, the appropriate strength box is emphasized with
    red (<5>characters),
    yellow (6-9 characters),
    or green (10+ characters) background colors.
  • When a level box is colored, the lower level is NOT. So, if the user gets to a strong level password the weak and medium are both grayed out.
  • As the user enters in their password in the second box, the password is checked with each keyup. The match feedback has a redbackground until the match is true. When the match is true, the background changes to green.
  • When the user submits the form and the passwords are matching and strong, the form submits as it should. If there is a problem with the password being unmatched or weak (medium is ok), the form process is broken.
